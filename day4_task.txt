1. Learn difference between <b> (bold styling) and <strong> (semantic importance). Practice both in sentences.

The <b> tag is used to make text bold for styling purposes only. It does not indicate any importance.
The <strong> tag also makes text bold, but it shows that the text is important and has semantic meaning for browsers and search engines.

code:

2. Understand <i> (italic styling) vs <em> (emphasis). Use them in a paragraph.


The <i> tag is used to display text in italic style without adding any special meaning.
The <em> tag is used to emphasize a word or sentence, indicating importance or stress.


code:

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>question2</title>
  </head>
  <body>
    <p>
    Learning HTML is <i>interesting</i> and helpful for web development.
    Students should <em>practice coding every day</em> to improve their skills.
    </p>
  </body>
</html>



3. Practice underlining text with <u>, highlighting with <mark>, and making side notes with <small>.



<u> is used to underline text.

<mark> is used to highlight important text.

<small> is used for side notes or less important text.


code:

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>question3</title>
  </head>
  <body>
    <p>This is <u>underlined text</u>.</p>
    <p>Please read the <mark>important instructions</mark> carefully.</p>
    <p>Submit the assignment today <small>(before 5 PM)</small>.</p>
  </body>
</html>


4.  Create hyperlinks using <a>. Add external links, internal links, and “back to top” links.


code:

    <title>question1</title>

  <body>

    <!-- Internal Link -->
    <a href="about.html">Go to About Page</a>

    <!-- Back to Top Link -->
